# linecard

文本标记：

```
    ----:横线
    [left]靠左
    [right]靠右
    [center]居中
    [pixel 400]指定像素
    [font size = 50,name = simsun,color = red,highlight = yellow]指定文本格式
    [style **kwargs] 控制参数
        height: 行高
        width: 行宽
        color: 本行颜色
    [nowrap]不换行
    [passport]保持标记
    [autowrap]自动换行
    [noautowrap]不自动换行
```
